# PRD — Weather Widget
*weship.today · April 2026*

## What is this?
A minimal weather widget that displays the current temperature, time, date, and location over a mood-driven illustrated background that changes based on weather conditions.

## Who is it for?
Anyone who wants a beautiful, at-a-glance weather check — form over function, but still functional.

## What does it do?
- Fetches live weather data from Open-Meteo (free, no API key required)
- Displays: current temperature, current time, today's date, city name, country
- Background illustration changes to match weather conditions: clear, cloudy, rainy, snowy
- User can type any city to get that location's weather
- Toggle between °C and °F

## What does it not do?
- No hourly or multi-day forecast (for now)
- No wind, humidity, or UV data (for now)
- No saved locations or history
- No automatic location detection via GPS

## Visual direction
Dark, moody illustrated landscapes. Grainy texture. Minimal white typography. The illustration carries the emotional tone — the data is secondary.
